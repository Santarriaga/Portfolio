Pest Control

About the application

Android application written in Kotlin. Made for a local pest control company. This app helps their team manage scheduled appointments. The app uses firebase for users authentication and firestore for its databse. It uses kotlin Flow and coroutines to fetch and monitor changing data.

The app uses Material io for a clean gui. It also uses Navigation component to traverse through fragments and for handling the navigation drawer. Has lottie library for animating the splashscreen. It uses intents to launch google maps and make phone calls.
