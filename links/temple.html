Temple

About the application

Android application written in Kotlin. Shopping app prototype meant to simulate purchasing clothes through mobile app. User can browse selection and view product details. Purpose was meant to experiment with material io databases, and coroutines.

App uses Firebase to store product information and serves as its database. It also uses a navigation drawer with Navigation Compose so that users can seemlessly travers through activiies.Uses Glide library to load images into recycler view and Coroutines for better performance.
