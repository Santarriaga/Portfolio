Canary

About the application

Android application written in Kotlin. Made with human safety in mind the app sends an automated message whenever the phone is shaked. User inputs contacts they wish to be send it beforehand. The message can be customized but for the most part it alerts the recipient that there is suspicious activiy around them and gives them the user's location.

The app uses room database to save contacts information and the customized message. Uses Navigation compose to traverse through fragments and for handling bottom navigation view. Has lottie library for animations and google maps to display users current location.
