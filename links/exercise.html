Excercise Planner

About the application

Android application written in Java.This phone application lets users search through various exercise and lets the add them to their weekly schedule. User can specify on what date they want to do that excerise and for how long.Each exercise has a category and can be found using search bar or can be displayed by category.

The app has a navigation drawer that lets user navigate through easily through each activity. Uses Glide library to load images into image views and recycler views. For its database it uses Room datebase to store and retrieve excerise details and the user's schedule.
