Canary

About the application

Android application written in Flutter. This application uses NASA API to retrive images of space. Every day it displays new picture of space and provides some information of the image. User can save them to view at a later time. It also has a date picker so you can view the image of a specific date or the user can use the explore feature which loads random images of space from NASA database.

The app uses Api called Astronomy Picture of the Day to make network calls and retrive images. It also use firebase to store user accounts and images that they liked. Has a navigation drawer for easy navigation. You can view the app in action here:https://youtu.be/zRgZklua6ss
